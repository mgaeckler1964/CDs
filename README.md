# CDs
compact disc database

dieses Programm entstand im Rahmen eines Schülerpraktikums Mitte der 90er Jahre.

Es sollte eine Verwaltung für Audio-CDs erstellt werden. Nachdem der 
Schüler wieder weg war, habe ich noch die Abspielfunktionen implementiert, damit 
ich die CDs mit dem Computer abspielen kann. Seit dem ich meine CDs aber mit 
iTunes eingelesen habe und sie als mp3s vorliegen, verwende ich diese Programm 
nicht mehr.


In dbTables.zip befinden sich die Datenbanktabellen. Mit ein paar Übungsdaten.
